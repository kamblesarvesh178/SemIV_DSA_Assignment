## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/kamblesarvesh178/SemIV_DSA_Assignment.git
   ```
2. Compile and run C++ files:
   ```bash
   g++ filename.cpp -o output
   ./output
   ```
3. Run Python files:
   ```bash
   python filename.py
   ```
